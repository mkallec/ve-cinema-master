# 映栈剧场 Vercel 母版

这是一个 Vercel 静态影视内容站母版，包含首页、全部内容、分类频道、最新更新、搜索、详情页、关于页和公共资源。

## 页面

- `index.html` 首页
- `movies.html` 全部内容
- `categories.html` 分类频道
- `updates.html` 最新更新
- `search.html` 搜索页
- `detail.html` 详情页
- `about.html` 关于/免责声明

## 后续批量生成

复制项目后，主要改这些：

- 站点名称
- SEO 标题和描述
- `assets/js/app.js` 里的内容数据
- `data-framework` 框架类型
- `assets/css/style.css` 的颜色变量
- 页脚版权文案

## Vercel 部署

项目是纯静态站，Vercel 可直接部署。
